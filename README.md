# 🎬 Netflix Data Analysis Project

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix
dataset to uncover insights about movies and TV shows available on
Netflix.

The analysis includes: - Distribution of Movies vs TV Shows - Top
genres - Top countries producing content - Content release trends over
the years - Rating distribution - Duration analysis
